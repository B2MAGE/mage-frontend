# MAGE Frontend

Frontend for the MAGE project, built with React, TypeScript, and Vite.

Right now this app is a simple landing page that points to the live demo while the broader platform is still in development.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Scripts

- `npm run dev` starts the local development server
- `npm run build` creates a production build
- `npm run preview` previews the production build locally
- `npm run lint` runs ESLint

## Current status

- Landing page for the MAGE platform
- Live demo link: `https://bsiscoe.github.io/MAGE/`
